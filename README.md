# Study Hanja
This project is an interactive site for the beginner levels of the national hanja test in South Korea, 한자능력검정시험, built for both Korean L2s and L1s.

## Purpose
In the last decade, resources for intermediate learners to study Sino-Korean vocabulary by using hanja have disappeared. This is a resource for Korean L2s at the intermediate level to help with building Sino-Korean vocabulary through hanja, which will also support Korean L1s studying for hanja proficiency tests.

## Tools and Technologies

- **VS Code** - I chose this as my code editor because I had experience with it already.
- **Netlify** - I chose this for hosting because it has a very simple process for hosting and had all the features I need.
