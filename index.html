A note on notation: if the sound is followed by :, the vowel is (was) long, and if it is followed by (:), it was optionally long. In modern Korean, vowel length is not marked.

Basic hanja

  • 8급

    • Only 50 here! Basic characters used in lots of common and easy vocabulary.

  • 7급II

    • Just 50 more, making 100 total. Like level 8, these are used in lots of common and easy vocabulary.

  • 7급

    • The last of the beginner level of basic daily use hanja. This level makes 150 total characters.

  • 6급II

    • The 75 characters in this level begin the intermediate level daily use hanja characters, making 225 total.
    • Up to now, only reading has been tested. From this point, the test includes writing questions as well. This level tests writing characters only from 8급.

  • 6급

    • This level has 75 characters, making 300 total.
    • Starting at this level, there are homonym questions. That means it will ask things like "which of these has the same sound but a different meaning?"
    • Also, you'll need to be able to write all characters up to 7급.
